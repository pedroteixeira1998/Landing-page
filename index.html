O que é a Comunidade CoDe?

A Comunidade CoDe é a comunidade exclusiva dos devs participantes da mentoria Conquiste Sua Vaga.

Comandada por Pedro Marins e Henrique de Andrade, acontecem diversas atividades para proporcionar desenvolvimento técnico e pessoal para todos os participantes.
